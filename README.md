### 常见问题：

Q:博主动态和归档是如何设置的？

A:

归档页面：后台创建独立页面，自定义模板选择[文章归档]。

博主动态：后台创建独立页面，自定义模板选择[说说动态]。成功创建后，前台访问该页面，可以进行发布动态。发布框只有博主登录的情况下才会显示。

Q：博客网站怎么设置博客信息？

A：在后台-控制台-外观，启用该主题后。然后到[侧边菜单栏-Echo-外观设置]栏，进行设置。
